# Viva Questions and Answers

## 1. What is a Library Management System?

A Library Management System is a software application used to manage books, students, and library transactions efficiently.

---

## 2. Why did you choose this project?

I chose this project to understand web development concepts, database design, and software project management.

---

## 3. What technologies are used in this project?

Frontend:
- HTML
- CSS
- JavaScript

Backend (Future):
- PHP

Database:
- MySQL

---

## 4. What are the main modules?

- Administrator Module
- Student Module
- Book Management Module
- Transaction Module

---

## 5. What is a database?

A database is a collection of organized data that can be stored, managed, and retrieved efficiently.

---

## 6. What tables are used in the database?

- Books
- Students
- Transactions

---

## 7. What is the purpose of the Transactions table?

It stores information about issued and returned books.

---

## 8. What are the advantages of this system?

- Faster book management
- Better record maintenance
- Reduced paperwork
- Improved efficiency

---

## 9. What are the future enhancements?

- Online Book Reservation
- QR Code Integration
- Mobile Application
- Email Notifications

---

## 10. What did you learn from this project?

I learned frontend development, database design, project documentation, GitHub usage, and software development practices.
