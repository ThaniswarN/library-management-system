# Project Summary

## Project Title

Library Management System

---

## Project Description

The Library Management System is a web-based application designed to manage books, students, and library transactions efficiently. The project aims to simplify library operations and reduce manual record keeping.

---

## Objectives

- Manage library books
- Maintain student records
- Track issued and returned books
- Improve library efficiency

---

## Technologies Used

### Frontend

- HTML
- CSS
- JavaScript

### Database Design

- MySQL

### Version Control

- GitHub

---

## Modules

### Administrator Module

- Add Books
- View Books
- Issue Books
- Return Books
- View Transactions

### Student Module

- Search Books
- View Borrowed Books
- View Profile
