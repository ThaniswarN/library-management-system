# Library Management System

## Introduction

The Library Management System is a web-based application developed to manage library operations efficiently. It helps administrators and students perform book-related activities in an organized manner.

---

## Objectives

- Manage books efficiently
- Maintain student records
- Track issued and returned books
- Improve library operations

---

## Modules

### Administrator Module

- Add Books
- View Books
- Issue Books
- Return Books
- View Transactions

### Student Module

- Search Books
- View Profile
- View Borrowed Books
- Return Books

---
