# Project Limitations

## Current Limitations

### 1. No Real Database Connection

The project currently uses a database design only and is not connected to a live MySQL database.

### 2. No User Authentication

Admin and Student login functionality is not connected to a backend system.

### 3. Static Data

Books, students, and transactions are displayed as static information.

### 4. No Online Access

The project is designed as a frontend prototype and does not support real-time online operations.

### 5. No Email Notifications

Users do not receive automatic notifications for due dates or book returns.

## Future Improvements

- Connect with MySQL database
- Implement PHP backend
- Add user authentication
- Add online reservation system
- Enable email notifications
