## Running the Project

### Method 1: Local Browser

1. Download the project.
2. Open the frontend folder.
3. Open index.html in a web browser.

### Method 2: GitHub

1. Upload files to GitHub.
2. Enable GitHub Pages (optional).
3. Access the website online.
