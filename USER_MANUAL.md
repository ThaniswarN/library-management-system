# User Manual

## Introduction

The Library Management System helps manage books, students, and transactions efficiently.

---

## Administrator Guide

### Login

1. Open the system.
2. Select Admin Login.
3. Enter username and password.

### Functions

- Add Books
- View Books
- Issue Books
- Return Books
- View Transactions
- Manage Students

---

## Student Guide

### Login

1. Open the system.
2. Select Student Login.
3. Enter username and password.

### Functions

- Search Books
- View Borrowed Books
- Return Books
- View Profile

---
