# Problem Statement

The college library currently manages book issuance and returns manually. This leads to problems like:
- Difficulty in tracking available books
- Time-consuming process for students and staff
- Errors in record keeping
- No proper search facility for books

### Proposed Solution
We are developing a **Library Management System** using React.js for frontend, Spring Boot for backend, and MySQL database. This web application will allow students to search and borrow books easily and help the admin to manage the library efficiently.
