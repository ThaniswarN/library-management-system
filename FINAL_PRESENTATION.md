# Final Presentation Notes

## 2-Minute Introduction

Good morning/afternoon everyone.

My project is Library Management System.

The purpose of this project is to manage library activities such as book management, student records, issuing books, and returning books in an efficient manner.

The system contains separate modules for administrators and students. It also includes book management, transaction management, and supporting documentation.

The project was developed using HTML, CSS, and JavaScript. The database design was prepared using MySQL concepts for future implementation.

Thank you.

## Project Objectives

- Maintain student records

- Reduce manual work
